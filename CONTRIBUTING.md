# Contributing

## Current status

Public contributions are not yet accepted because this repository contains a licence placeholder. The owner must approve the final licence and contribution terms before enabling contribution workflows.

## After contributions are enabled

A useful proposal should identify the exact page, explain the reader need, provide a current primary source where relevant, disclose the contributor's relationship to the subject, and contain no confidential or personal information. Maintainers may edit wording for accuracy, neutrality, consistency, and scope.

Do not use public issues for credentials, private data, client details, unpublished vulnerabilities, or other sensitive material. Follow [SECURITY.md](SECURITY.md) for security reporting.
